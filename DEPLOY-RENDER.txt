Render — ClipFlow Stripe API
=============================

1) Push the repo to GitHub (includes stripe-backend/ and render.yaml).

2) Backend Web Service — Root Directory

   Dashboard: your backend service → Settings → Build & Deploy → Root Directory

   Set to:  stripe-backend

   Build:   npm install
   Start:   npm start

   (If you use the Blueprint from render.yaml, rootDir: stripe-backend should match this.)

3) Environment (same service → Environment):

   STRIPE_SECRET_KEY = sk_...
   STRIPE_PRICE_ID = price_...
   CHECKOUT_SUCCESS_URL = https://your-frontend.example.com/success
   CHECKOUT_CANCEL_URL = https://your-frontend.example.com/cancel
   STRIPE_WEBHOOK_SECRET = whsec_...

4) Save and redeploy.
