Render Web Service — use these EXACT settings
================================================

1) Root Directory  (Settings → Build & Deploy)
   MUST be:  stripe-backend

   This is the folder that contains this API’s package.json and src/server.js.
   Do NOT leave Root Directory empty unless you use the repo root package.json shim.

2) Build Command:
   npm install

3) Start Command:
   npm start

4) Health Check Path:
   /health

5) Environment variables (Environment tab):
   STRIPE_SECRET_KEY = your sk_test_... or sk_live_... key
   (Also set STRIPE_PRICE_ID, CHECKOUT_SUCCESS_URL, CHECKOUT_CANCEL_URL, STRIPE_WEBHOOK_SECRET, etc.)

6) Save, then Manual Deploy (or push to Git if auto-deploy is on).

If you see wrong paths or missing stripe-backend, Root Directory is not set to stripe-backend.
